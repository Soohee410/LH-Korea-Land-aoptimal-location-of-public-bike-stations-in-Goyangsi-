# LH Optimal-Locations-of-Public-Bike-Stations

고양시 공공자전거 스테이션 최적지 선정 대회

### 목적

 - 공공자전거 운영이력 데이터 및 공간 데이터를 활용하여 자전거 스테이션의 최적 위치를 선정하여 향후 시민들의 공공자전거(공유자전거) 사용에 대한 접근성 개선.
 
### 대회 일정

- 2020.07.28 ~ 2020.09.11

팀: 두부와 숲이 (박기찬, 박수희)

**최종순위**: 1등(최우수)
